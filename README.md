# 个人技术博客部署 

#### Install hexo
```
npm install hexo-cli -g
```
#### Setup your blog
```
hexo init blog
cd blog
```

#### Start the server
```
hexo server 或者简写成 hexo s
```

#### Create a new post
```
hexo new "Hello Hexo"
```

#### Generate static files
```
hexo generate
```

#### Setup my theme
```
vim _config.yml 
change theme into hexo-theme-archer
```

#### clone theme
```
cd themes\hexo-theme-archer 
git clone https://github.com/horizon2019/hexo-theme-archer.git
```
